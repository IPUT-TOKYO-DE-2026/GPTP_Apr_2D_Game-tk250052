#  Bounce Shot

## ゲームの内容
操作できるプレイヤーから弾を発射して、敵を狙い撃つ。
壁や天井などを使って弾を反射させ、速いタイムを目指そう。

<img width="1525" height="759" alt="スクリーンショット 2026-04-27 175351" src="https://github.com/user-attachments/assets/35872bfd-d8df-43e0-aa27-81bb05258f89" />


## 操作説明
- AとDキーを使って横に移動する
- スペースキーを使って弾を発射する
- 矢印キー左と右を使いプレイヤーの角度を変える
